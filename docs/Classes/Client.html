  • update(type:id:body:completionHandler:)

    Update data of the provided unique id (GET request)

    Declaration

    Swift

    public func update(type: String, id : String, body : [String : Any], completionHandler: @escaping (Any?, Any?) -> ())

    Parameters

    type

    Type of data that is created in the app (Appbase dashboard)
    id

    ID of query
    body

    JSON structured data parameter that has to be passed for updating, Note: For updating data, the JSON must be of the format doc{ JSON FOR THE PARAMETER TO BE UPDATED }. Eg : let updateParameters:[String:Any] = [ doc: [ year: 2018 ] ]

           While updating, all the JSON body needs to be put inside a doc array as shown above else the method won't work.
       For more information : [https://www.elastic.co/guide/en/elasticsearch/reference/2.4/docs-update.html#_updates_with_a_partial_document](https://www.elastic.co/guide/en/elasticsearch/reference/2.4/docs-update.html#_updates_with_a_partial_document)

    Return Value

    JSON response and the error occured if any in format (Any?, Error?)

  • bulk(type:body:completionHandler:)

    Make bulk requests on a specified app or a specific type. Bulk requests can be any of index, update and delete requests.

    Declaration

    Swift

    public func bulk(type: String? = nil, body : [String : Any]? = nil,completionHandler: @escaping (Any?, Any?) -> ())

    Parameters

    type

    Type of data that is created in the app (Appbase dashboard), should only be passed if you want to make the request to the that perticular type.
    body

    JSON structured data parameter that has to be passed for updating, Note: For updating data, the JSON must be of the format {request_type:{JSON}} : { index: { _type: users, _id: 2 } } { doc : {field2 : value2} } { delete: { _id: 2 } } For more information : https://www.elastic.co/guide/en/elasticsearch/reference/2.4/docs-bulk.html

    Return Value

    JSON response and the error occured if any in format (Any?, Error?)
